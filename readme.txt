Elementor add-on voor Hipsy Events Core. Voegt Hipsy event widgets en dynamic tags toe aan Elementor.

== Description ==

Hipsy Events for Elementor voegt Elementor widgets en dynamic tags toe voor websites die Hipsy events tonen via Hipsy Events Core.

Met deze plugin kun je Hipsy events visueel tonen en stylen in Elementor, waaronder eventtitels, data, tijden, locaties, beschrijvingen, afbeeldingen, tickets, ticketknoppen, zoek- en filtermogelijkheden en reviews.

== Requirements ==

* WordPress 6.0 of hoger
* PHP 7.4 of hoger
* Elementor
* Hipsy Events Core

== Installation ==

1. Upload de pluginmap naar `/wp-content/plugins/` of installeer de plugin via het WordPress plugin-scherm.
2. Activeer eerst Hipsy Events Core.
3. Activeer Elementor.
4. Activeer Hipsy Events for Elementor.
5. Open Elementor en gebruik de Hipsy widgets of dynamic tags.

== Frequently Asked Questions ==

= Heb ik Elementor Pro nodig? =

Nee, de widgets werken met Elementor. Voor sommige Theme Builder workflows kan Elementor Pro handig zijn.

= Heb ik Hipsy Events Core nodig? =

Ja. Deze plugin is een Elementor add-on voor Hipsy Events Core en gebruikt de events en metadata die door de core plugin beschikbaar worden gemaakt.
